# Machine Learning
Machine Learning
## 项目简介
本项目是机器学习和深度学习的代码实现部分。
### matrix.cpp
* 手写 Vector 和 Matrix 结构体，支持运算符重载。
* 实现了线性可分数据的随机生成逻辑。
* 支持多维空间（如 4 维、10 维）的分类测试。
* 包含毫秒级精度性能计时器。
在低维海量数据（N=10000, D=2）下，原始形式远快于对偶形式。

### XOR-preceptron.cpp
* 利用ReLU函数优化感知机，实现对线性不可分的分类问题的初步实现。
* 利用随机梯度下降法对二位异或分类问题实现了成功解决
